galib
=====

Genetic Algorithms library written in Java

### Usage
    java -jar GAlib.jar <algorithm> <input-file>

### List of Algorithms
 - TGA: Simple Genetic Algorithm

### Input file for SGA
A .properties file with the folowing keys
 - P_c 
 - P_m
 - M
 - G
 - L
 - Func
 - Verbose (verbose level)
